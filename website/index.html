About Zodiac

Zodiac is a domain-specific programming language designed specifically for generating personalized zodiac sign summaries based on user-provided birth data. Created by DaniaA, this language simplifies the process of collecting user birth information and outputting structured astrological insights.

Whether you're building astrology applications, personality analysis dashboards, chatbots, or educational tools, Zodiac provides a clean and intuitive syntax to work with astrological data.

Features

Zodiac Sign Type

Built-in ZodiacSign type with enums for all 12 signs makes working with astrological data intuitive and type-safe.

Date Processing

Automatic date parsing and time-zone aware calculations ensure accurate astrological calculations.

Template Literals

Easily combine user data with zodiac descriptions using template literals for clean and readable code.

Functional Paradigm

First-class functions allow for flexible and modular code organization.

Standard Library

Comprehensive library for input parsing, date utilities, and zodiac data calculation.

Developer Tools

VSCode extension with syntax highlighting, code snippets, and inline documentation.

FizzBuzz Implementation

// FizzBuzz Program in Zodiac language
func main() {
    print("FizzBuzz Program");
    print("---------------");
    
    var limit: String = input("Enter the maximum number: ");
    var max: Int = parseInt(limit);
    
    runFizzBuzz(max);
}

func parseInt(str: String) -> Int {
    // Simple parsing function
    return str;
}

func runFizzBuzz(max: Int) {
    for (var i = 1; i <= max; i = i + 1) {
        if (i % 15 == 0) {
            print("FizzBuzz");
        } else if (i % 3 == 0) {
            print("Fizz");
        } else if (i % 5 == 0) {
            print("Buzz");
        } else {
            print(i);
        }
    }
}

main();

Installation

Getting started with Zodiac is simple:

# Clone the repository
git clone https://github.com/user/zodiac-language.git

# Navigate to the project directory
cd zodiac-language

# Install dependencies
pip install -r requirements.txt

# Run a sample program
python run_zodiac.py examples/hello_world.zod

Requirements:

  • Python 3.8+
  • textX
  • PyTZ (for timezone calculations)
